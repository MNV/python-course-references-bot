References Bot
==============

Чат-бот предназначен для автоматизации процесса формирования списка использованных источников
по указанному стандарту цитирования посредством чат-бота для Telegram.

Функции чат-бота
----------------

Чат-бот автоматически оформляет список использованных источников по указанному
стандарту цитирования на основе переданных ему входных данных.
Для заполнения входных данных чат-бот предоставляет пользователю интерфейс
с возможностью заполнения данных об источниках.

После добавления информации об использованных источниках пользователь может загрузить
заполненный файл и получить от чат-бота файл в формате Word с оформленным в соответствии
с выбранным стилем цитирования списком литературы.

В чат-боте предусмотрен интерфейс с кнопочным меню для возможности взаимодействия с пользователем.
